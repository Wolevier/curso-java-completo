# Estudos de Java

Repositório dedicado aos exercícios e projetos do curso, focado em consolidar a lógica de programação e fundamentos da linguagem.

## Organização do código
Os códigos estão organizados na pasta `src`, divididos pelos tópicos do curso (seções de estrutura sequencial e condicional if-else).

## Tecnologias
* Java
* IntelliJ IDEA
