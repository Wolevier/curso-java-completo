### Estudos de Java - Programação Orientada a Objetos

Este repositório guarda os exercícios e projetos práticos que desenvolvo durante meus estudos da linguagem Java e dos pilares da Programação Orientada a Objetos (POO).

### 🚀 Tecnologias utilizadas

*   Java (JDK 25)
*   IntelliJ IDEA

### 📁 Estrutura do projeto

O código está organizado dentro da pasta `src`, separado por tópicos:

*   `secao04_estrutura_sequencial`: Exercícios iniciais de lógica, entrada/saída de dados e operações matemáticas.
